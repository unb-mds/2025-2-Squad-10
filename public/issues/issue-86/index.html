[FEATURE] FAzendo a responsividade da pagina inicial

[FEATURE] FAzendo a responsividade da pagina inicial

Descrição

Atualmente, a Homepage e seus componentes (Hero, Equipe, Sobre, Footer, etc.) possuem um layout fixo otimizado apenas para desktop. Precisamos adicionar Media Queries para garantir que o site se adapte corretamente a tablets e dispositivos móveis.

Objetivos

  • Ajustar tamanhos de fonte no Hero para evitar quebras de texto em telas pequenas.
  • Transformar o Grid da seção Equipe (3 colunas) em 1 coluna no mobile.
  • Reorganizar o layout do Footer para empilhar colunas verticalmente em telas menores.
  • Ajustar espaçamentos (padding/margin) nos componentes Sobre o Projeto e Como Funciona.
  • Garantir que o Grid de Features se comporte de forma fluida (wrap) ou coluna única.

Arquivos Afetados

  • Hero.css
  • Equipe.css
  • SobreProjeto.css
  • Footer.css
  • HomePage.css
  • SobreNos.css
  • ComoFunciona.css
