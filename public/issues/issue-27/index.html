📝 Descrição / Objetivo

Configurar os ambientes de desenvolvimento para o projeto, garantindo que tanto o frontend quanto o backend possam ser executados de forma isolada e padronizada entre os membros da equipe.

✅ Tarefas

  • Frontend

    • Criar o ambiente com Vite + React
    • Documentar como inicializar o projeto no frontend

  • Backend

    • Inicializar o projeto
    • Instalar dependências principais

📌 Critérios de Aceitação

  • Frontend e Backend rodam em ambientes independentes, cada um com seus próprios node_modules.
  • Instruções de instalação e execução documentadas na pasta doc
  • Ambos os ambientes podem ser iniciados com um único comando em seus respectivos diretórios.
